These are the modifications:

-A Splash screen.

-An Options form. You have to fill this form in order the program works properly.

-Now you can view the last e-mail you imported in a separate form.

-You can view each e-mail you have import, sorted by year, month and date, delete its contents from the database or exported it in a text file.

-You can import directly from the internet (via a small e-mail client) the PSC e-mails. Here I use the FreePOP ActiveX control from Glenn Harm (http://www.harm.net/FreePOP). At this point I use an e-mail rule in order the e-mail client to read only the Planet Source Code e-mails. You can change this if you want from the Options form.

-You can import directly from Outlook or export the Outlook e-mails in separate text file for each e-mail. The code was originally written from Kristof Denayer. I only made some changes. All the credits go to him.

-I change the look a little bit. I add a toolbar in the main form. I also change the UI in the frmImport form. I use the FolderView control from CCRP (www.mvps.org\ccrp). You can find some good staff there. The code which give me the date from each file comes from API-Guide, a program that contains at this point description and examples for 577 API functions. You can download it free from www.allapi.net.

-You can send directly to me the program errors by pressing the e-mail button in the Error Form. You have to be connected with the net in order to send me the e-mail. At this point I use the vbSendMail.dll from http://www.freevbcode.com/code/vbSendMail.zip.

-I add trace code in order to know how you got to the error. The trace source code is from an article from Nikolai Paradizov titled «Building an Application Error Handler» published in DevX on Apr 13 2000. I include the article in the zip file. I change it a little bit in order to meet my needs.

    I think that someone can learn a lot of things from this program. It covers a lot of different areas (Database programming, error handling, ActiveX controls like TreeView, ListView and ToolBar, internet programming etc).
